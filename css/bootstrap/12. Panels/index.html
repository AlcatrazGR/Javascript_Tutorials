Bootstrap - Panels tutorial

Bootstrap panels are basically bordered boxes with some padding inside of them. On panels you can also have footer or header panels with different colours. Always use the class .panel to define that a div will be a panel. Each panel can be consisted of 3 divs and onemptied fourth that bind all together, with the classes :

  • panel-heading -> The heading of the panel.
  • panel-body -> The main body of the panel.
  • panel-footer -> The footer of the panel.

Contextual Panels

The same way as the rest of elements in bootstrap there are classes with different colours that define the purpose of the message such as :

  • .panel-default -> Simple panel no colours.
  • .panel-primary -> Panel with blue colour.
  • .panel-success -> Panel with green colour that is used for success messages.
  • .panel-info -> Panel with light blue colour that is used for info messages.
  • .panel-warning -> Panel with yellow colour that is used for warning messages.
  • .panel-danger -> Panel with red colour that is used for danger messages.
