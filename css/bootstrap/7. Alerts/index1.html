Bootstrap Alerts

As with the coloured table rows that we saw in a previous example the same idea is used in alerts. There are danger alerts that point out an error, there are the info alerts that point out an info message box, the warning alerts to warn for something and lastly the success alerts to show a successful process.



There is the main class .alert that is always used to define that the div is an alert message box followed by either :

  • alert-success
  • alert-info
  • alert-warning
  • alert-danger

Success! Indicates a successful or positive action.
Info! Indicates a info action.
Warning! Indicates a warning action.
Danger! Indicates a non successful or bad action.

You can also have alert message boxes with a closing 'X' button on them just by using an a tag inside the alert and giving to it the attributes :

  • class="close"
  • data-dismiss="alert"
  • aria-label="close"
There is no need to use any javascript for the closing process. From the moment you use the above three attributes the 'X' can close the alert.
