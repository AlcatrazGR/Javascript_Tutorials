Bootstrap provides multiple different text tags that make the web page more dynamic and beautiful.

Bootstrap provides sub title texts by using the tag 'small' inside the header (h1...h6) tags. The bigger the header tag the bigger the sub text will be. Ofcourse u can change the size of the sub text manually using CSS styling and 'font-size' attribute.

Bootstrap also provides the 'mark' tag to mark text inside your web page
Bootstrap provides the tag 'abbr' to show a point of description. For example if in your page you use arconyms for companies or organisation ( HTML ) you can use this tag to explaing to the user the definition of the acronym. By mouse over the area a popup will appear with the description of the acronym. The description of the acronym is placed inside the 'title' attribute of the abbr tag.
Bootstrap provides blockquote tag. The blockquote element is used to present content from another source:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
From WWF's website
The source of the quote inside the blockquote tag is defined through the 'footer' tag. As blockquote exists in bootstrap also blockquote-reverse exists which flips horizontally the text to the other side.
For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

!Important, The blockquote-reverse is NOT a tag element, but a class which is defined inside the blockquote tag.
