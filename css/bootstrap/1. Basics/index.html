In order for a bootstrap page to work properly also in mobile phones and be adaptive there are two class selectors from bootstrap qhich are :
  • 'container' --> Which provides a responsive fixed width container which will bind the data of the web page.
  • 'container-fluid' --> Which provides a full width container spanning to the entire width of the viewport.

Your web page must follow one of those two options of containers in order to work properly to mobile pages.

! Important, Don't forget that the container tha you will choose must be the outer one who will bind the whole page.

! Important, Bootstrap containers are not nestable, meaning that you cant put a container inside another...

You will notice by using bootstrap and the containers that the styling of each HTML element inside the bootstrap container is different from the simple HTML tags. Thats because the bootstrap library changes the styling of all HTML objects...
