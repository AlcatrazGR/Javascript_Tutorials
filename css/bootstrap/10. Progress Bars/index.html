Bootstrap - Progress Bars Tutorial

Bootstrap provides a number of different progress bars for usage, each having its own colour, text and even animation. Progress bars are used to show the progress of a certain task to the user.

Warning !, Progress bars are not supported in Internet Explorer 9 and earlier (because they use CSS3 transitions and animations to achieve some of their effects).
Info !, Ofcourse in order to make a bootstrap progress bar function and really display the progress of a task in real time then you need to use javascript.
