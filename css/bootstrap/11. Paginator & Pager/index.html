Bootstrap Pagination & Pager Tutorial

If you have a web site with lots of pages, you may wish to add some sort of pagination to each page. Another good practice to use pagination or pager is through database results, assuming that we have a webpage that displays a list of items for example 100 items. Having a huge list like that in one page and the user scrolling all the way continuously to see all the items is not optically appealing, so in those cases we can user pagination or pager.
