Bootstrap Carousel

Carousel plugin is a component that is used to cycle between objects such as images like a carousel (slideshow).

Warning !, Carousel plugin is not supported by internet explorer 9 orphans earlier versions.

In order to achieve the carousel effect you need to user the classes .carousel and the .slide in a div element in order for bootstrap to understand that the element is a carousel.

The basic components of a carousel effect is the ol element that contains all the navigation dots, the div element with the class .carousel-inner which contains other div elements each one for each image and lastly the two a elements for the left and right controls.

Warning!, For better use it is good to use the same size of images, the reason is that the carousel object will take the size of the currently displayed image, so basically the whole structure of page will change when the images change.
