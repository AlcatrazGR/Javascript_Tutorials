Collapse Items

In bootstrap we can make certain items collapse such as panels text with the press of a button and more without the use of javascript or jquery.

Button Text Collapse

By using two html attributes inside the button element data-toggle="collapse" data-target="#demo" we can define that way that the button will trigger a collapse event and it will be on the element with id demo which is the element that holds that info to be collapsed.
