Bootstrap - Tooltips & Popovers

Tootips

Tooltips are small message-boxes that appear to the user when he hovers on a object. Tooltips can be applied by putting two parameters in a element the data-toggle and the title . The first parameters takes the value tooltip to define that the particular object has a tooltip text and the title is basically the text that will appear on the tooltip. After putting all the tooltips in the required elements the only thing necessary is to enable them through javascript or jQuery.

$(document).ready(function(){
       $('[data-toggle="tooltip"]').tooltip();
});

You can also define the placement of a tooltip by simply using the attribute data-placement with values : top, right, bottom, left.

Popovers

Popovers are similar to tooltips, they are basically a pop up box that appears when the user clicks on a element. The difference is that the popover can contain more content and can be styled. The way to create a popover is to use the data-toggle and giving it the value of popover. Also in order to use a popover at the end we need to enable all of them through javascript or jQuery.

$(document).ready(function(){
       $('[data-toggle="popover"]').popover();
});
