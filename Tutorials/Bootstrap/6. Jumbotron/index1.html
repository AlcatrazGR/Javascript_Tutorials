Bootstrap Jumbotron Tutorial

Jumbotron is a class provided by bootstrap which indicates a big box for calling extra attention to some special content or information. Inside a jumbotron you can put text but also valid HTML elements and even elements with bootstrap classes.

Jumbotron Outside The .container Class

As you can see you can also use a .jumbotron outside the .container div. The difference between the two is that outside of the .container the jumbotron will stretch in all of the page's width than the normal .jumbotron inside a .container.
