Bootstrap Images

Bootstrap provides a number of class selectors in order to change the appearance of an image. It can make a image more rounder, with radius or even as a thumbnail.

  • Basic Image Classes
  • Rounded Corners : uses the class .img-rounded
  • Circle : uses the class .img-circle
  • Tumbnail : uses the class .img-thumbnail

Responsive Images

Responsive images uses the class .img-responsive in order to change the size of an image dynamically when the size of the browser page changes.

Resize the page and see the results ...

Image Gallery

By using bootstrap grid with the .img-thumbnail class one can achieve an image gallery.

Responsive Embeds

You can embed videos, slide shows with bootstrap that will dynamically change their size when the size of the browser changes. In order to do that we use the classes .embed-responsive and .embed-responsive-16by9 in a div that will bind the iframe element and inside the iframe element we use the class .embed-responsive-item to describe that the particular item is a responsive item.
