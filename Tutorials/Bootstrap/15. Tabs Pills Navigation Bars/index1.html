Navigation Bars Tutorial

Navigation bar is an item used on the top of a page and it provides the main navigation. Its implementation is similar to pills and tabs but with different classes. Here we use the element nav and inside it the standard class .navbar to define that the item will be a navigation bar and there are a number of other classes to define the type of the navigation bar which are :

  • .navbar-default -> The standard navigation bar with grey colour.
  • .navbar-inverse -> An alternative black navigation bar.
Warning !, Whatever type of navigation bar you choose to implement bear in mind that in small screen such as mobile phones the navigation bar will be shown in a vertical order and not horizontally.

Classic

This navigation bar contains a header with the use of the class .navbar-header inside a div (you dont need to use a header its optional). The rest of the menu items are basically a ul element which uses the classes .navbar-nav and .nav
