Bootstrap - Buttons

Bootstrap buttons come in different colours, different sizes and with many other functionalities. Bootstrap button classes can be used either in `a` tags or `button` tags or `input` tags. In buttons we always use the class .btn to define that the object will be a button and then one of the classes listed below.

Button Types

  • Class .btn-default
  • Class .btn-primary
  • Class .btn-success
  • Class .btn btn-info
  • Class .btn btn-warning
  • Class .btn btn-danger
  • Class .btn btn-link

Button Sizes

  • Class .btn-lg
  • Class .btn-md
  • Class .btn-sm
  • Class .btn-xs

Button Block

A block button is a button that uses the class .btn-block and it takes full width of the content that its in.

Button Activated / Disabled

Bootstrap class that enables or disables a button. If the button is disabled it changes it style to a more transparent colour and also on mouse hover a disabled icon is displayed.
