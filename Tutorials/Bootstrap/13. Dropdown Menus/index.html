Dropdown Menus

Bootstrap provides a number of classes to create and handle a dropdown menu without the use of jquery or javascript.

A simple dropdown menu uses the class .dropdown to define that the object will be a dropdown menu. Inside the div there are two necessary elements a button element that we give the classes : .btn , .btn-primary , .dropdown-toggle which define that it is a primary button and that it will toggle from top to down, and the second element is a span that we give to it the class .caret which creates the down arrow. The rest of the elements used in the div is basically a ul to create all the links.
