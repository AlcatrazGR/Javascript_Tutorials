Bootstrap Forms

In bootstrap we define an area which is a form by using the class form-control . Each form element with that class has a 100% width. There are three types of forms and how they align with the group, those are :

  • .form-group
  • .form-inline
  • .form-horizontal

Input States

Input fields can have different states on which they operate, those states are :

  • INPUT FOCUS - The outline of the input is removed and a box-shadow is applied on focus
  • DISABLED INPUTS - Add a disabled attribute to disable an input field
  • DISABLED FIELDSETS - Add a disabled attribute to a fieldset to disable all controls within
  • READONLY INPUTS - Add a readonly attribute to an input to prevent user input
  • VALIDATION STATES - Bootstrap includes validation styles for error, warning, and success messages. To use, add .has-warning, .has-error, or .has-success to the parent element
  • ICONS - You can add feedback icons with the .has-feedback class and an icon
  • HIDDEN LABELS - Add a .sr-only class on non-visible labels

Input Sizing

Input elements can be sized using specific bootstrap classes. You can set the height of an input field by using the following :

  • .input-lg
  • .input-sm
You can also set the width of an input element with the following :
  • .col-lg-*
  • .col-sm-*
Important !, The more splited into smaller parts a bootstrap grid is the more smaller the width of the elements will become.
