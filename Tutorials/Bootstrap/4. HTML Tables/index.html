Bootstrap Tables

Bootstrap provides an amount of classes that will style a HTML table element. In every table class the compiler will try to fit the table as best as it can in the page and the columns to have the appropriate width.

The .table class

The .table class is the classic class selector that bootstrap provides for table elements. It will style a table making only visible the rows.

!Important, In order for each and every class selector to work in a table there must always be the class .table inside in order to be bootstrap compatible...

The Contextual Classes

The contextual classes use the simple .table class, but in every row a class of either :

  • success -> For displaying a success message.
  • danger -> For displaying a error message.
  • info -> For displaying a info message.

The Responsive Tables

The responsive tables uses a class .table-responsive inside a div element that will then bind the whole table. That way for small devices the user can scroll horizontally the table without the compiler making the columns too small in order to fit to the actual screen.
