Grid Layout

Bootstrap greed layout example ...

Bootstrap grid layout it is made using div tags and in each row it supports 12 columns max. Bootstrap also provides different types of classes to define the grid to which object it will refer to, such as :
  • xs --> Class that is used to build grids for phones
  • sm --> Class that is used to build grids for tablets
  • md --> Class that is used to build grids for desktops
  • lg --> Class that is used to build grids for larger desktops

Now, in every grid layout we define a row using the class="row" and each column has a class like the following :

class="col-*-*"

Where the first asterisk is where we put the type of grid class (xs, sm, md, lg) and the second asterisk is used to define the number of columns it will use.

!Important, The number of columns used in every row much be of maximum number of 12, no more no less...
