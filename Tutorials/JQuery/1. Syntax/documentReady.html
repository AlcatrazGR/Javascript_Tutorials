In JQuery the $(document).ready(function(){ ... }) event is widely used and inside of it are all the JQuery methods. This used to prevent any JQuery code to b excecuted before the document (the web page) is fully finished loading. It is good practice to wait for the page to be fully loaded so there wont be any mistakes - fails at the page. Such fails can for example be to try hide a element that is not created yet or changeing the size of an image that hasnt been loaded to the page.
