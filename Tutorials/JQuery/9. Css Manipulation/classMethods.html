It is well known that JQuery uses code to access HTML and also CSS elements and change their atributes. In this example we will demonstrate the usage of three usefull methods on JQuery with which we can add / remove css classes for an HTML object. We can initialize .class selectors to change the css style of an HTML object and the with JQuery code add or remove those to DOM Objects.

With addClass method we can add a class CSS selector to a certain or more DOM objects. In the above example we add a class to the paragrapgh to change its color and a class to the div to change its background color. With removeClass method we can remove a class that it is binded to a DOM object. In the example above we remove the classes that was previously manually added to the paragraph and div objects. With toggleClass method we can switch to adding or removing certain class declaration binded to a DOM object.
