JQuery provides three very useful functions in order to access an HTML object's attributes. Those function are, the text(), html(), val() functions. With text() function we can access the text of an object. In this example we access the text of three 'p' elements in order to set them the texts Empty Field and Successful login. With the html() function we can access the html code of an object. Lastly the val() function returns the value of an html object, in this example we get the values of the tou inputs to check if any of those two is empty.
