Simple Usage
The simplest usage o v-select, using a v-select element and a one dimension array as an option.
Multiselect Usage
Vue-select library allows for an easy and stress less usage of multiselectio. This is achievable by using the attribute `multiple` on the v-select element.
Tagable Usage
Taggable is a usefull attribute of vue-select that lets you select an option event if it does not exist. That is achievable through the search input.
Custom Usage
You can easily create a custom select element by using templates and slots.
Custom With Images Usage
Also in a custom select element you can use images on options and whatever other HTML element.
