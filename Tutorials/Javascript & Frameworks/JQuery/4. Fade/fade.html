Fade Function Examples

The fadeIn function makes visible a previously hiden html object. It can take up to two parametres which the first one is the delay of the fadeIn and the second is a name of a function which we want to be excecuted when the object is fully fadded in. Both of those parametres are optional. The above image has a fadeIn delay of 3500 millisecond. The fadeOut function makes a previously visible html object to be hidden. It can take up to two parametres which the first one is the delay of the fadeIn and the second is a name of a function which we want to be excecuted when the object is fully fadded in. Both of those parametres are optional. The above image has a fadeOut delay of 3500 millisecond. The fadeToggle switches an object to visible and hidden status. If an object is visible then it will become hidden else if the object is hidden then it will become visible again. The above image has a fadeToggle function with fade delay of 3500 milliseconds. The fadeTo function makes an object hidden to a certain opacity value. The value can be between 0 and 1. The value 0 is referes to completely faded object while 1 as completely visible. This function can take up to three parametres, 1st the duration, 2nd the opacity value, 3rd a function name whose code we want to be excecuted when the object has been completely faddedTo.
