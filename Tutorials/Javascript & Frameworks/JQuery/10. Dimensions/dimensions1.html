height / weight dimensions. 120 x 120 dimensions
Weight ang Height methods in jequery return the dimensions of a DOM object. It does not include any other attribute such as borders, padding etc.




innerWidth / innerHeight dimensions. 120 x 120 dimensions with 15 pixel padding.
InnerWidth and InnerHeight in JQuery return the dimensions of a DOM object along with any padding that is refered to the object. In this example there is a div object with 120 x 120 dimensions and 15 padding. That means that the current dimensions of this object is 120 + (15 (padding left) + 15 (padding wright)) = 150 height and 150 width (150 x 150).




outerWidth / outerHeight dimensions. 120 x 120 dimension with 2 pixel border and 15 pixel padding.
outerWidth and outerHeight in JQuery return the dimensions of a DOM object along with any padding or border that maybe refered to the object. In this example there is a div object with 120 x 120 dimensions and 15 padding and 2 pixel border. That means that the current dimensions of this object is 120 + (15 (padding left) + 15 (padding wright)) + (2 (pixel top) + 2 (pixel bottom)) = 154 height. The same goes for width.
