Angular JS Directives

Directives are used to add functionality to an application. Angular has a set of built in directives for more info press Here
Directives are extended HTML attributes that start with the `ng` prefix.

Custom Directives
You can create your own custom Angular JS directives and program their response by using the .directive function...
To inovoke a custom directive you can use on of the following methods :
  • Element Name -> Create a new HTML element with the name of the directive.
  • Attribute -> Set the custome directive as an attribute (what angular does by default)
  • Class -> Then class class of the elements is the directive itself
  • Comment -> HTML comment can initialize a directive like directive: this-is-a-custom-directive
To initialize a custom directive with the first way (`Element Name`) you need to set a HTML tag with its words seperated by `-` <`this-is-a-custom-directive`>. But whent its time to call it through the .directive function you need to use camel case name like .directive("thisIsACustomDirective", function() ...)
