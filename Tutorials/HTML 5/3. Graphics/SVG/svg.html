Scalable Vector Graphics (SVG)

SVG is a new element added to HTML 5. Like canvas it provides a number of methods to draw paths, circles, texts and other graphics into the web page. The main difference between canvas and svg is that svg does not require any use of javascript code all its methods are basically html parameters.
