This is the first HTML 5 example. In HTML 5 there are two specific declaration that must be used in every html document. Those declarations are DOCTYPE html which declares that this specific document is an html document and meta charset="UTF-8" this declaration is used for the document to recognize every type of character even if its from an another language than english.

In HTML 5 many knew tags has been added and others have been removed. Tags that have been added are :
  • New semantic elements like header, footer, article, and section.
  • New form control attributes like number, date, time, calendar, and range.
  • New graphic elements: svg and canvas.
  • New multimedia elements: audio and video.

Tags that have been removed in HTML 5 are :
  • acronym
  • applet
  • basefont
  • big
  • center
  • dir
  • font
  • frame
  • noframes
  • strike
  • tt
All those tags/elements are either replaced by another new tags or are now changed by CSS code and not HTML.
HTML 5 also provides a new set of API's which are :
  • HTML Geolocation
  • HTML Drag and Drop
  • HTML Local Storage
  • HTML Application Cache
  • HTML Web Workers
  • HTML SSE
