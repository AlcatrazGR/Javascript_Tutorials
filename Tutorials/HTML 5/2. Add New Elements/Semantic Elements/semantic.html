This is the structure of the new semantic elements added to HTML 5.
HTML 5 provides us with new semantic elements. Usually with normal HTML we used div elements to determine the parts of the page. Now in HTML 5 there are new elements such as header article, aside, section, footer which determine the parts of the page. Of course in order for he new semantics to be for example 500px wide we need to change its CSS attributes.
