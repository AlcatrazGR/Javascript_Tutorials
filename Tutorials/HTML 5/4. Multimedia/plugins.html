What are Plug-ins
Plug-ins or also known as helper applications (HTML helpers) they are used in order to extend the functionality of the HTML browser. Basically they are computer programs that are added into a web page to fulfill a certain task because HTML documentation doesn't have any apropriate elements for this task. Examples of well known plugins are the Java applets display maps, scan viruses and more.
Plug-ins Usage
Plug-ins are added inside a web page using the object or embed element / tag.
Object Element
The object element is supported by all the browser and italic defines an embedded object inside the HTML web page. We can use the object element in order to embed Java applets, PDF readers, Flash player and more.
Embed Element
The embed element is supported by in all the major browsers such as Chrome, FireFox etc. Because embed element wasn't supported by previous editions of HTML such as HTML 4 then its only usable in a HTML 5 web page.
