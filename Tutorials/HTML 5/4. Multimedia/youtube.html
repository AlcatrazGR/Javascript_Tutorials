YouTube Videos
On the multimedia videos we have seen how to embed a video inside our web page using the HTML 5 element video. There another easier and less time consuming way by using YouTube videos instead.
Advantages Of YouTube Videos
  • Less time consuming.
  • You don't have to convert your video multiple times in order to be recognized by a wide variety of web browsers.
  • Faster web page load. The reason that we have an embedded video inside our web page means that the compiler of the page need time to display it.
  • Less burden for your server. An embedded video needs to be located with the files of the web page so it is necessary to upload it into our server in order to be found by the compiler that is hard space consuming. On the other hand youtube videos doesn't consume any space on your server.

YouTube Parameters
There are some parameters that can be used inside the URL link of the youtube video in order to achive a certain goal. Those parameters are : autohide, autoplay, controls, loop and playlist. The first four parameters can take a value which determine their state for example the parameter autoplay if it is setted with value 0 then that means that the video wont play automatically when the web page is fully loaded. For more information on the parameters and their values you can read the documentation of W3CSchools ( YouTube Parameters ).
Object & Embed Elements
We can achive the same goal of embedding a youtube video in our web page by using the object element or even the embed element. The only thing that changes is that we don't use the src attribute as we used inside the iframe element but we use a new attribute called data.
